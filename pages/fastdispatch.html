FastDispatch Emergency Training Simulator

FastDispatch is an emergency dispatch training simulator built to help users practice high-pressure decisions through realistic scenarios, timed prompts, and AI-assisted feedback.

View on Devpost Emergency Write-up Resume Portfolio Home

What it does

FastDispatch simulates realistic emergency calls and asks the user to make decisions under time pressure. The system guides trainees through scenario flow and provides feedback so they can improve response accuracy and confidence.

What I built

  • Scenario flow system (prompt → choices → results → next step)
  • Timed interactions to simulate real dispatcher pressure
  • AI-powered explanation feedback for incorrect answers
  • UI logic to keep the experience fast and readable

Tech / Tools

  • Python
  • Google Gemini API (AI explanations + feedback)
  • UI + scenario logic

Why it matters

Dispatch decision-making is high-stakes and time-sensitive. This project focuses on practice and repetition: learning the right questions to ask, recognizing urgency, and responding clearly under pressure.
